📌 Project Overview

This Power BI project analyzes sales data to find insights about revenue, profit, customers, products, and delivery performance.
The dashboard helps users understand business performance using interactive visualizations.

📊 Features
Total Sales Analysis
Profit Analysis
Customer City Insights
Product Category Performance
Delivery Status Tracking
Interactive Filters and Slicers
Monthly Sales Trends
🛠 Tools Used
Power BI
Microsoft Excel / CSV Dataset
Data Cleaning
Data Visualization
📂 Dataset Information

The dataset contains:

Order ID
Customer Details
Product Category
Quantity
Sales Amount
Delivery Status
Order Date
📈 Dashboard Visuals
Bar Charts
Pie Charts
Line Charts
KPI Cards
Tables & Slicers
🎯 Objective

The main objective of this project is to convert raw sales data into meaningful business insights for better decision-making.

🚀 How to Open
Download the .pbix file
Open it using Microsoft Power BI Desktop
Refresh the dataset if needed
